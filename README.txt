Зато работает!

      Инструкция ебать!:
В list копируешь строки с названиями (адреса не нужно, а то будет жопа),
ну и exe-шник шмёшь. Ну и забираешь готовый iptvchannels.m3u
Всё, как бы.

     Так, инфа, если интересно:
source/source.txt - это исходный файл, качается каждый раз при запуске экзешника.
iptvchannels.m3u - генерируется каждый раз при нажатии экзешника.
list.txt - его меняешь только ты и вручную.

    Проблемы:
- Если канал пропадает из плейлиста-источника - ссылка на него в стирается с плейлиста-выхода.
- Если нет файла "list.txt" всё вылетает.
